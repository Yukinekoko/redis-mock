##### 目前 eval 指令存在的问题:

1. 在lua脚本中可以创建全局变量（在redis中执行的lua脚本不允许创建全局变量，会返回异常）
2. 在lua脚本中支持了package基础包的相关函数（在redis中执行的lua脚本没有引入package包）
3. 在lua脚本中尚未支持第三方包struct、cjson、bitop、cmsgpack
4. 暂不支持在lua脚本中调用redis.log指令
